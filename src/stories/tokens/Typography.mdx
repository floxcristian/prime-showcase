# Tipografía

**16px** es el tamaño base en todo el chrome de la aplicación —
body, nav, inputs, buttons, menús, labels, breadcrumbs, list items.
ERP en español + sesiones de 8h = legibilidad le gana a densidad.

La fuente es **Inter Variable**, self-hosted desde `@fontsource-variable/inter`
para evitar el CDN de Google Fonts.

## Combinaciones aprobadas

Cada combinación más abajo es canónica. Cualquier cosa fuera de esta lista
pasa por code review.

```text
TÍTULOS
  text-3xl font-bold     leading-normal    ← hero (text-3xl SIEMPRE bold, enforced por ESLint)
  text-2xl font-medium   leading-8         ← sección
  text-xl  font-medium   leading-7         ← subsección (headers de inbox / movies)

SUBTÍTULOS
  text-muted-color font-medium leading-normal   ← subtítulo de página
  text-color       font-bold   leading-6        ← título de card

LABELS
  text-color font-semibold leading-6   ← label de input (arriba del control)
  text-color font-normal   leading-6   ← label de checkbox/radio (al lado del control)

LINKS
  font-medium text-primary hover:text-primary-emphasis underline cursor-pointer transition-colors duration-150

BODY / NAV
  text-color leading-6                  ← body por defecto
  font-medium leading-6                 ← nav item (L1, L2, breadcrumb)
  font-semibold leading-6               ← nav activo / último crumb del breadcrumb

PEQUEÑO
  text-xs font-medium                   ← badges / contadores / pills
  text-xs leading-4                     ← emails / sub-labels / timestamps

ESPECIAL (solo contenedores estrechos)
  text-sm font-medium leading-tight     ← movie titles
  text-sm font-medium leading-none      ← labels de avatar overlay
```

## Distribución de pesos

| Peso | Uso |
|---|---|
| `font-medium` (500) | DEFAULT — ~85% del uso |
| `font-semibold` (600) | Labels de input, nav activo, último crumb del breadcrumb |
| `font-bold` (700) | Título de card, hero `text-3xl` |
| `font-normal` (400) | Solo labels de checkbox / radio |

Enforcement:

- `showcase/text-3xl-requires-bold` — hero text-3xl debe ser `font-bold`.
- `showcase/label-requires-semibold` — labels de input deben ser `font-semibold`;
  labels de checkbox / radio deben ser `font-normal` (y nunca `font-semibold`).
- `showcase/no-forbidden-typography` — bloquea tamaños / pesos fuera de escala.

Ver `Primitives/Button`, `Recipes/Page Header`, `Recipes/Form Card` para ejemplos en vivo.
