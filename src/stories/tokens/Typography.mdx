# Typography

**16px** is the base size across the entire chrome of the application —
body, nav, inputs, buttons, menus, labels, breadcrumbs, list items.
Spanish-first ERP + 8h sessions = legibility wins over density.

The font is **Inter Variable**, self-hosted from `@fontsource-variable/inter`
to avoid the Google Fonts CDN.

## Approved combinations

Each combination below is canonical. Anything outside this list goes through
code review.

```text
TITLES
  text-3xl font-bold     leading-normal    ← hero (text-3xl ALWAYS bold, ESLint-enforced)
  text-2xl font-medium   leading-8         ← section
  text-xl  font-medium   leading-7         ← subsection (inbox / movies headers)

SUBTITLES
  text-muted-color font-medium leading-normal   ← page subtitle
  text-color       font-bold   leading-6        ← card title

LABELS
  text-color font-semibold leading-6   ← input label (above the field)
  text-color font-normal   leading-6   ← checkbox/radio label (beside the control)

LINKS
  font-medium text-primary hover:text-primary-emphasis underline cursor-pointer transition-colors duration-150

BODY / NAV
  text-color leading-6                  ← body default
  font-medium leading-6                 ← nav item (L1, L2, breadcrumb)
  font-semibold leading-6               ← nav active / breadcrumb last crumb

SMALL
  text-xs font-medium                   ← badges / counters / pills
  text-xs leading-4                     ← emails / sub-labels / timestamps

SPECIAL (narrow containers only)
  text-sm font-medium leading-tight     ← movie titles
  text-sm font-medium leading-none      ← avatar overlay labels
```

## Weight distribution

| Weight | Usage |
|---|---|
| `font-medium` (500) | DEFAULT — ~85% of usage |
| `font-semibold` (600) | Input labels, nav active, breadcrumb last crumb |
| `font-bold` (700) | Card title, `text-3xl` hero |
| `font-normal` (400) | Checkbox / radio labels only |

Enforcement:

- `showcase/text-3xl-requires-bold` — hero text-3xl must be `font-bold`.
- `showcase/label-requires-semibold` — input labels must be `font-semibold`;
  checkbox / radio labels must be `font-normal` (and never `font-semibold`).
- `showcase/no-forbidden-typography` — blocks off-scale sizes / weights.

See `Primitives/Button`, `Recipes/Page Header`, `Recipes/Form Card` for live examples.
