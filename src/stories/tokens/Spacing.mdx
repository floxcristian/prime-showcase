# Spacing

Una escala, tres familias. Cada gap / padding / margin del chrome
debe resolver a uno de estos valores. Las violaciones están bloqueadas por
`showcase/no-forbidden-spacing`.

## Escala

<div style={{ marginTop: '1rem', fontFamily: 'Inter Variable, system-ui, sans-serif' }}>
  {[
    { name: '1', px: 4 },
    { name: '2', px: 8, note: 'gap por defecto' },
    { name: '3', px: 12 },
    { name: '4', px: 16, note: 'padding por defecto, mt/mb por defecto' },
    { name: '5', px: 20 },
    { name: '6', px: 24, note: 'cards, secciones' },
    { name: '8', px: 32, note: 'excepción solo de chat' },
  ].map(({ name, px, note }) => (
    <div key={name} style={{ display: 'flex', alignItems: 'center', gap: '1rem', padding: '0.25rem 0' }}>
      <div style={{ width: '4rem', textAlign: 'right', fontFamily: 'monospace', fontSize: '0.875rem' }}>
        {name} · {px}px
      </div>
      <div style={{ background: '#0074c2', height: '1rem', width: `${px}px`, borderRadius: '2px' }} />
      {note && <div style={{ color: '#52525b', fontSize: '0.875rem' }}>{note}</div>}
    </div>
  ))}
</div>

## Reglas por familia

```text
GAP     gap-1 | gap-2 ← DEFAULT | gap-3 | gap-4 | gap-5 | gap-6 ← CARDS | gap-8
PADDING p-1 | p-2 ← DEFAULT | p-3 | p-4 | p-6 ← CARDS | px-4 py-1 (buttons) | px-7 py-5 (header)
MARGIN  mt-1 (sutil) | mt-2/mb-2 | mt-4/mb-4 | mt-6/mb-6 | mb-0
FLEX    grow | shrink-0 (avatares) | flex-1
```

## Prohibido

- `gap-9` y más arriba (excepto los documentados `gap-7`, `gap-8` de chat).
- `p-7`, `p-8` fuera de chat (`py-8`) y del marketing panel de login (`px-12`).
- `m-7+` (sin excepciones documentadas).
- Cualquier spacing arbitrario como `gap-[13px]`, `p-[20px]`, `m-[1rem]`.

## Excepciones documentadas

Patrones preexistentes aceptados en code review y registrados en
`AUDIT_BASELINE.md`:

| Clase | Dónde | Por qué |
|---|---|---|
| `p-5` | side-menu | Entre p-4 y p-6 para la densidad de navegación |
| `gap-7`, `gap-8`, `py-8` | chat | Aire para el área de mensajes |
| `px-12` | panel de marketing del login | Spacing tipo hero estilo Stripe/Linear |
| `p-[1px]` | indicador de badge | Alineación pixel-perfect del overlay badge |
| `mt-3`, `mt-5`, `mb-5`, `mt-10` | layouts / chat media | Spacing fino preexistente |

Agregar una nueva excepción requiere:
1. Documentar el caso de uso en la descripción del PR.
2. Agregar a `ALLOWED_EXCEPTIONS` en `tools/eslint/rules/no-forbidden-spacing.js`.
3. Linkear al rationale en `AUDIT_BASELINE.md`.

Ver `Recipes/Page Header`, `Recipes/Form Card`, `Recipes/Multi-Panel Layout`
para spacing en composición real.
