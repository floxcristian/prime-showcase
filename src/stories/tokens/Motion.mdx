# Movimiento

**Animar el mínimo de propiedades en el mínimo tiempo.** El
usuario debería notar el cambio, no la animación. Las transiciones internas
de PrimeNG están deshabilitadas en runtime (`transitionDuration: '0s'` en
`app.preset.ts`) para evitar el theme-flash durante navegación client-side;
todas las transiciones reales viven en clases narrow de Tailwind en el template.

## Tokens

```text
DURATION
  0ms     PrimeNG theme swap (token de runtime, no es una propiedad CSS)
  150ms   transition-colors / transition-opacity (narrow por default)
  250ms   transition-transform en overlays (sheets, panels)

EASING
  cubic-bezier(0.2, 0, 0, 1)   estándar (default Aura)
  cubic-bezier(0, 0, 0.2, 1)   decelerate (entradas)
  cubic-bezier(0.4, 0, 1, 1)   accelerate (salidas)
```

## Propiedades de transition permitidas

| Propiedad | Clase | Cuándo |
|---|---|---|
| Color / fondo | `transition-colors` | Default para elementos interactivos |
| Opacity | `transition-opacity` | Imágenes, avatares con hover por opacidad |
| Movimiento | `transition-transform` | Carousel, sheets, panels |
| Eje específico | `transition-[transform]` | Overrides narrowly-scoped |

## Prohibido

- ❌ `transition-all` — anima toda propiedad que cambie, esconde jank por reflow.
  Bloqueado por `showcase/no-forbidden-transitions`.
- ❌ Bare `transition` — mismo problema, scope ambiguo.
- ❌ Animar `width`, `height`, `top/left/right/bottom`, `margin`,
  `padding` — gatillan layout, causan reflow, se ven con jank.

Usar `transform: translate*()` + `opacity` para movimiento. Los cambios
de posición vía transform son GPU-composited; las propiedades de layout no.

## `prefers-reduced-motion`

El proyecto respeta el setting del sistema del usuario. Cualquier animación debe vivir
dentro de:

```scss
@media (prefers-reduced-motion: no-preference) {
  /* animaciones acá */
}
```

CI emula `prefers-reduced-motion: reduce` en una configuración de Playwright suite.
Los screenshots con motion forzado a 0 deben ser estables
(sin pixel-jitter).

## Receta de duration

El único lugar donde se declara `duration-*` explícitamente es la receta
canónica de link — elementos `<a>` de texto:

```html
<a class="font-medium text-primary hover:text-primary-emphasis underline cursor-pointer transition-colors duration-150">
  Texto del enlace
</a>
```

Para todo lo demás: omitir `duration-*` y dejar que aplique el default de Aura (que
ya respeta el motion token).
